CSS Hamburger Examples

HTML5 Boilerplate 5.3.0 Template

CSS Burger code provided by Jonathan Suh - https:/github.com/jonsuh/hamburgers/blob/master/dist/hamburgers.css

I chose to use plain JavaScript instead of jQuery to handle the click event and toggle the CSS class.

  1. Retrieve all the divs that I will want to toggle the CSS class on.
  2. Loop through the array of elements to add an eventListener to each
  3. Use the toggle() method on classList to add or remove the 'is-active' class
    An issue I had was that the 'is-active' class was being added to the 'hamburger-inner' div element
    This was resolved by using e.currentTarget as opposed to e.target
